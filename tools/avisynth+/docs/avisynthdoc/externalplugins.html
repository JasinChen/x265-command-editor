Avisynth 2.5 Selected External Plugin Reference¶

General info¶

AviSynth plugins are external linked modules (libraries) implementing one or more additional functions (filters) for deinterlacing, noise reduction, etc. A great number of plugins (200 or more) were written by independent developers as free software.

An up-to-date list of external plugins is available here.
