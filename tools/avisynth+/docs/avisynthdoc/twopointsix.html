AviSynth 2.6¶

New Video Features¶

  • A new color formats: YV24 (24 bit), YV16 (16 bit), YV411 (12 bit) and Y8 (8 bit; greyscale). For a description of the color formats AviSynth can deal with see here.

New Audio Features¶

  • none

New Internal Features¶

  • MultiThreading support, including ScriptEnvironment::SetMTMode and ScriptEnvironment::GetMTMode.
